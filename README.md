# ML Weather App 🌦️

A desktop weather application built with Python that fetches real-time weather information using the OpenWeatherMap API. The application also predicts the next temperature using a simple Linear Regression model and displays the results in a graph.

---

## Features

- Search weather by city name
- Live weather information
- Current temperature
- Feels like temperature
- Humidity
- Wind speed
- Sunrise and sunset timings
- Temperature prediction using Linear Regression
- Graphical visualization using Matplotlib
- Simple desktop interface built with Tkinter

---

## Technologies Used

- Python
- Tkinter
- Requests
- NumPy
- Scikit-learn
- Matplotlib
- OpenWeatherMap API

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ml-weather-app.git
```

Move into the project folder

```bash
cd ml-weather-app
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```

## How It Works

1. Enter the city name.
2. The application requests weather data from the OpenWeatherMap API.
3. Current weather information is displayed in the application.
4. Forecast data is used to train a simple Linear Regression model.
5. The model predicts the next temperature.
6. A graph compares the current temperature, feels-like temperature, humidity, and predicted temperature.
